Memory Game


Instrucciones

1) Al Iniciar, saldra una palabra la cual se eliminara en cierto tiempo.

2)Debes Indicar que palabra era, si aciertas subieras de nivel si fallas perderas una vida.

3)Existen 3 niveles, cada que avances de nivel el tiempo de la palabra en pantalla se reducira.

4)Solo tienes 3 vidas.
